# Quillo

A minimal blogging platform where users can write, edit, and discover posts.

## Features

- Sign up and log in with JWT authentication
- Create posts with a rich text editor (Quill)
- Add a cover image by uploading a file (stored via Cloudinary)
- Edit and delete your own posts
- View all posts from the community
- View your own posts separately

## Tech Stack

**Frontend**

- React
- Material UI
- React Router
- React Quill
- Axios
- Cloudinary (cover image hosting)

**Backend**

- Node.js
- Express
- SQLite (better-sqlite3)
- JWT (jsonwebtoken)
- bcrypt
